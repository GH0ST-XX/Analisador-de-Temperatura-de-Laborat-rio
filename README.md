# Este projeto foi desenvolvido como um teste de conhecimento prático para a disciplina de **Programação I** do curso de **Engenharia de Computação**

O programa recebe valores de temperatura por terminal, classifica cada leitura em categorias (Alta, Baixa ou Padrão) e gera um relatório com as médias de cada faixa sem o risco de quebrar o terminal caso alguma lista fique vazia.

##
O que foi usado no código
* **Python 3**
* **Lógica aplicada:** Estrutura condicional encadeada (`if/elif/else`), tratamento preventivo de erros (validação em linha para evitar `ZeroDivisionError`), funções com múltiplos retornos e manipulação de listas.
##

Funcionalidades

- Coleta dinâmica de dados inseridos pelo usuário.
- Classificação automática em três níveis: Alta (+50°C), Baixa (-0°C) ou Padrão.
- Tratamento de erros para entradas inválidas.
- Cálculo de médias aritméticas precisas por categoria.
- Relatório final descritivo e estatístico.

  ##
Como rodar o projeto
1. Certifique-se de ter o Python instalado.
2. Execute o script pelo terminal ou VS Code: analisador_temp.py
